## Create virtual environment to isolate your dependencies
```
python -m venv venv
```

## Activate vitual environment
```
venv\Scripts\Activate
```

## Install dependencies
```
pip3 install httpx
```

```
pip3 install selectolax
```

```
pip3 install requests
```
